NotiqueBot

AI-powered TikTok automation with GPT & n8n

About This Project

NotiqueBot is an experimental tool that uses GPT-generated ideas and automates TikTok video creation and posting using n8n workflows. It is designed for educational, non-commercial use and runs on a local or cloud-based setup.

How It Works

  • AI (GPT) generates creative video ideas
  • n8n triggers video assembly and upload
  • Videos are posted to TikTok via the official API
